# User Stories för Läslistan

## 1. Visa tillgängliga böcker
Som användare vill jag kunna se en lista med alla tillgängliga böcker så att jag kan välja en att lägga till på min läslista.

## 2. Lägga till en bok till min läslista
Som användare vill jag kunna markera en bok som favorit så att jag kan spara böcker jag vill läsa.

## 3. Lägga till en ny bok
Som användare vill jag kunna lägga till en egen bok genom att fylla i titel och författare, så att jag kan lägga till böcker som inte finns i katalogen.

## 4. Ta bort en bok från min läslista
Som användare vill jag kunna ta bort en bok från läslistan om jag inte längre vill läsa den.

## 5. Se enbart mina favoriter
Som användare vill jag kunna växla vy för att endast se mina favoriter.
